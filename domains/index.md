# Domains

Connect custom domains to your static sites and cloud instances.
Mincemeat handles SSL certificates automatically and provides DNS
validation guidance so you can point your domain to the right endpoint.

## What you can do

| Capability | What it means |
| --- | --- |
| Understand DNS records | Learn about CNAME and A records, TTL, and DNS propagation. |
| Connect a custom domain | Add a subdomain or apex domain and validate ownership through DNS. |
| Check domain status | Monitor validation, SSL provisioning, and active states. |
| Understand caching | Learn how Mincemeat caches content and how to purge the cache. |
| Troubleshoot domain issues | Diagnose validation failures, SSL problems, and DNS errors. |

## Getting started

If you are connecting a custom domain for the first time, start with
[DNS basics](/domains/dns-basics) to understand the record types you
will need, then follow
[Connect a custom domain](/domains/connect-custom-domain) to add your
domain.
