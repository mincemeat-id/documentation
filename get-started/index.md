# Get Started

Welcome to the Mincemeat user documentation. This section covers what
Mincemeat is, how to access your account, and the steps to get started
with your first resources.

## What is Mincemeat

Mincemeat is a platform for managing cloud instances and hosting static
sites. It provides a unified interface for deploying, monitoring, and
maintaining web applications without managing infrastructure directly.

## What you can do

- **Cloud Instances**: View assigned instances, monitor status and
  metrics, use the web terminal, and manage proxy settings.
- **Static Sites**: Host websites with upload or GitHub deployments,
  connect custom domains, and roll back to previous versions.
- **Headless CMS**: Create content collections and fields, manage media assets, and fetch dynamic schemas and entries via edge-cached REST APIs.
- **Domains**: Connect custom domains with automatic SSL and DNS
  validation.
- **Account**: Manage your password, two-factor authentication, and
  audit history.
