# Contact support

Contact support after you have checked the relevant troubleshooting
steps and still cannot complete the task. A concise request with safe
context helps support investigate faster.

## Before you start

- Copy any correlation ID shown with the error.
- Note the page, action, and approximate time of the problem.
- Remove secrets and unrelated customer data from screenshots or logs.

## What to include

| Include | Why it helps |
| --- | --- |
| Correlation ID | Lets support find the matching event. |
| Page or feature | Narrows the area to check. |
| Action you tried | Explains the workflow that failed. |
| Approximate time | Helps match logs and audit history. |
| Visible error message | Shows what Mincemeat reported to you. |
| What changed recently | Helps identify password, 2FA, domain, or deployment changes. |

## What not to include

- Passwords, authenticator codes, recovery codes, session cookies, or
  API tokens.
- Full browser logs without reviewing them for sensitive values.
- Screenshots that show unrelated account details or private customer
  information.
