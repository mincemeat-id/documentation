# Disable two-factor authentication

Disable two-factor authentication only when you understand the reduced
account protection. After disabling 2FA, your password is the only
required sign-in factor.

## Before you start

- You must be signed in.
- You need your current password.
- You need a current authenticator code.

## Steps

1. Open **Account Settings**.
2. Select the **Two-Factor Auth** tab.
3. In **Disable Two-Factor Authentication**, select **Disable 2FA**.
4. Enter your **Current Password**.
5. Enter your **Authenticator Code**.
6. Select **Confirm Disable**.

## What happens next

Mincemeat shows a confirmation message and removes 2FA from your
account. Your old recovery codes stop being useful because recovery
codes only apply while 2FA is enabled.

The change is recorded in **My Activity**.

## Troubleshooting

| Problem | What to check |
| --- | --- |
| **Confirm Disable** is unavailable | Enter both your current password and authenticator code. |
| The authenticator code is rejected | Wait for the next code and confirm device time is automatic. |
| You lost your authenticator app | Use a recovery code to sign in first, then contact support if you cannot complete the disable step. |
