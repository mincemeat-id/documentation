# Sign out

Sign out when you are done using Mincemeat, especially on a shared or
untrusted device. Signing out ends the current browser session.

## Before you start

- You must be signed in.
- Save any work in progress before leaving pages with forms or terminal
  sessions.

## Steps

1. In the app sidebar, select **Log out**.
2. Review the confirmation dialog.
3. Select **Log out** again.
4. Confirm that Mincemeat returns you to the sign-in screen.

## What happens next

Mincemeat clears the current browser session. Other browsers and
devices remain signed in unless you change your password with **Sign
out from all other devices** selected.

## Troubleshooting

| Problem | What to check |
| --- | --- |
| You closed the browser without signing out | Open Mincemeat again and use **Log out** if the session is still active. |
| You lost a device | Change your password and select **Sign out from all other devices**. |
| You are immediately signed in again | Confirm you are using the same browser profile and clear stale site data if the issue continues. |
