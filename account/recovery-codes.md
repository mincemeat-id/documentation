# Use recovery codes

Recovery codes let you sign in when you cannot access your
authenticator app. Each recovery code works once, so replace your codes
after using one.

## Before you start

- Your account must already have two-factor authentication enabled.
- You need one unused recovery code.
- If you are regenerating codes, you need your current password and an
  authenticator code.

## Sign in with a recovery code

1. Enter your email and password on the sign-in screen.
2. When **Two-Factor Authentication** appears, paste one unused recovery
   code in **Verification code**.
3. Select **Verify**.
4. After you sign in, go to **Account Settings**.
5. Select **Two-Factor Auth**.
6. Regenerate recovery codes so you have a complete current set.

## Regenerate recovery codes

1. Open **Account Settings**.
2. Select the **Two-Factor Auth** tab.
3. In **Recovery Codes**, select **Regenerate Codes**.
4. Enter your **Current Password**.
5. Enter your **Authenticator Code**.
6. Select **Regenerate**.
7. Save the new codes when **Save Your Recovery Codes** appears.
8. Select the acknowledgement checkbox.
9. Select **I've Saved My Codes**.

## What happens next

Regenerating recovery codes invalidates all previous recovery codes.
Only the newest set can be used for future sign-ins.

## Troubleshooting

| Problem | What to check |
| --- | --- |
| A recovery code is rejected | Confirm it has not already been used and that you copied the whole code. |
| You used your last recovery code | Sign in with it, then regenerate codes immediately. |
| You lost both the authenticator app and recovery codes | Contact support through your normal support channel. Do not send passwords or old recovery codes. |
